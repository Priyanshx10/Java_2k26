# 🎮 Pac-Man Game (Java Swing)

A fully functional **Pac-Man game built using Java Swing and AWT**, demonstrating core game development concepts like rendering, movement, collision detection, and event-driven programming.

---

## 🚀 Features

* 🧱 Tile-based map rendering system
* 🎮 Player (Pac-Man) movement using keyboard input
* 👻 Multiple ghost characters
* 🍒 Food collection system
* 🔁 Game loop using Timer and ActionListener
* 💥 Collision detection with walls
* 🖼️ Custom sprite rendering using Graphics API

---

## 🛠️ Tech Stack

* **Java**
* **Swing (javax.swing)**
* **AWT (Graphics, Event Handling)**

## ▶️ How to Run

### 1. Compile the project

```
javac *.java
```

### 2. Run the application

```
java App
```

---

## 📦 Create Runnable JAR

```
jar cfe PacMan.jar App *.class Images
java -jar PacMan.jar
```
